Project Rekall is a tile flipping and matching game.

Players have to find and match all 9 pairs of tiles in each level to extend the time limit and refresh all the tiles.

The more pairs of tiles players matched, the more score points they will acquire.

When players are matching 2 or more pairs in a row, a score multiplier will apply to the score.

When time runs out, the game will be finished and players will see their score.

Livestock Management System

Interface

This is a livestock management system that can read a database of animals, perform queries and generate report.

It reads data from Microsoft Access Database files (*.accdb)

When users type in an ID of an animal, details about this animal such as animal type, cost, weight will be shown in the text box.

It can generate an overall report of the farm, including total profitability, total tax paid to the government and so on.

It can calculate the profit of each animal in the farm, sort them by profit and generate a text file.

It can also calculate the ratio of animal below certain age among the whole farm.
